Despre acest site

Acest site s-a nascut din dorinta de centraliza informatii din mai multe despre acest subiectul debitarii cu laser, sau taiere cu laser. Informatiile prezentate aici sunt o colectie de date din surse cum ar fi Wikipedia, Enciclopedii, si alte surse online si offline.

Un alt motiv care face posibila existanta acestui site, este dorinta de a oferi o vedere de ansamblu asupra posibilitatile nelimitate care sunt ascunde in spatele acestor minuni tehnologice moderne, cum este laserul. Pe aceasta directie am incercat sa prezentam si o parte mai noua a taierii cu laser, numita de unii Steel Design, care dupa cum este prezentat si in pagina dedicata, este defapt o arta, mai mult decat o prelucrare industriala a materialelor.
